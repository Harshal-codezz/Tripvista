# TripVista

TripVista is a responsive travel booking web interface that brings flights, hotels, trains, buses, holiday packages, and limited-time travel deals together in one place. It is designed as a modern front-end project focused on helping users explore destinations, compare offers, and book travel services through a clean and engaging experience.

## Overview

TripVista presents itself as an all-in-one travel platform with support for multiple booking categories, featured deals, destination discovery, user testimonials, and app promotion sections. The interface highlights convenience, secure booking, price-focused offers, and quick access to travel planning tools.

## Features

- Multi-service booking interface for flights, hotels, trains, buses, and holiday packages.
- Hero search section with trip type options such as one-way, round-trip, and multi-city.
- Popular destinations showcase with pricing highlights for places like Goa, Manali, Rajasthan, Kerala, Andaman, Dubai, Bali, and Bangkok.
- Travel service cards covering domestic flights, international flights, hotels, train tickets, bus tickets, and holiday packages.
- Limited-time deals section for flights, hotels, train routes, and package offers.
- Trust indicators such as secure booking, instant confirmation, and best price guarantee.
- Traveler testimonials and mobile app promotion area.
- Footer sections for quick links, support resources, and contact information.

## Usage

After launching the project in a browser, users can:

- Browse available travel categories from the navigation bar.
- Search for flights using source, destination, departure date, return date, and traveller class fields.
- Explore curated destinations and promotional deals.
- View service highlights for hotels, trains, buses, and holiday packages.
- Read customer testimonials and access app-style travel features such as live tracking, price alerts, saved cards, and e-tickets.

### Example workflow

```text
1. Open the homepage.
2. Select Flights from the booking tabs.
3. Enter source and destination cities.
4. Choose travel dates and passenger details.
5. Click Search Flights.
6. Explore deals and destination sections below.
```
